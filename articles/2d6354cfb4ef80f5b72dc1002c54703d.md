## 概要



会社で Cursor 導入することになり、正月の重い腰を上げて試してみると、直感的な操作性とエージェントのレスポンス速度に驚きました。


VSCode を使い慣れていた私が、とりあえず Cursor 入門してみたメモです。


## Cursor とは?


Cursorは、コードベースを理解し、自然言語での指示でコーディングを効率化するAI搭載のコードエディターです。VSCodeから設定を移行できるため、既存のワークフローを維持しながら、AIの力を活用できます。


Cursorは以下の機能を提供します：

- コードベースを理解し、自然言語での指示でコーディングを効率化するAI搭載のコードエディター
- Tabでオートコンプリート
- 選択範囲をインライン編集
- エージェントとチャット
- ルール
- コードベースのインデックス作成 (LSP内蔵)
- モデル選択可能

## 環境の準備



VSCodeから移行する際、設定を手動で再現するのは面倒だと思っていましたが、Cursorには「Import from VSCode」機能があり、ワンクリックで設定を移行できました。

1. [https://cursor.com/ja](https://cursor.com/ja) でダウンロード
2. サインアップ、7日間の無料トライアル (Pro Trial)
3. Import from VSCode で設定移行


7日間の無料トライアルがあるので、気軽に試せます。ひとまず実際に使ってみて、エージェントのレスポンス速度が速いことに驚きました。今まで使っていた Claude Codeと比べても、体感的に2倍以上速く感じます。


## 基本設定



VSCodeからimportしたので、特に追加の設定は不要でした。キーバインドや拡張機能の設定もそのまま引き継がれたので、違和感なく使えています。


ただし、Cursor特有の機能（エージェントやタブ補完）については、少し慣れるまで時間がかかりました。


## タブ補完


タブ補完は、コードをオートコンプリートしてくれる機能です。オートコンプリート専用の「Tab」というモデルで動作しています。



タブ補完機能は本当に便利です。コードを書いていると、AIが次に書くべきコードを予測して半透明で表示してくれます。


### 使い方


半透明で表示されたら：

- Tab で受け入れ
- Escape で却下
- Cmd + → で単語単位で受け入れ


「Cmd + → で単語単位で受け入れ」は、長い補完を部分的に受け入れる際に便利です。全部受け入れると冗長な場合でも、必要な部分だけを選択できるので、細かい調整がしやすくなります。


## エージェント


Cmd + I でエージェントタブが開きます。コード編集、レビュー、CLI操作、ブラウザ操作等、プロンプト指示を出すと自律的にやってくれます。



エージェント機能は、Cursorの最大の魅力だと思います。コード編集、レビュー、CLI操作、ブラウザ操作など、プロンプトで指示を出すだけで自律的に作業を進めてくれます。特にブラウザ連携は、要素選択してのビジュアル編集や、スクショの読み込み、自動テストなど強力そうで、これから活用していきたい機能です。


### エージェントとは



(Cursorに限らず)そもそもエージェントとはなにか、今更ですがもう少し掘り下げたいと思い、公式ドキュメントを確認しました。本質的には、エージェントとは「ツールを反復的に呼び出す仕組み」にすぎません。つまり、AIが複数のツール（コード編集、ファイル操作、ブラウザ操作など）を順番に呼び出して、目的を達成する仕組みです。


参考: [Agents | Cursor Learn](https://cursor.com/ja/learn/agents)


## ショートカット



Cursor特有のショートカットを覚えると、作業効率がさらに上がります。特に「Cmd + I」でエージェントを開くのは頻繁に使うので、すぐに覚えられました。ショートカット一覧は公式ドキュメントに詳しく載っているので、必要に応じて参照しています。



Cursor 特有で便利そうなもの：

- **Cmd + I** エージェントを開く
- **Cmd + Shift + L** 新しいエージェントを開く
- **Ctrl + Tab** エージェントのセッション切り替え
- **Ctrl + C** チャットを中断
- **Shift + Cmd + B** ブラウザ開く
- **Opt + Cmd + '** 過去のチャット履歴を見る
- **Cmd + .** または **Shift + Tab** モードを切り替える
- **Cmd + /** AIモデル切り替え
- **Cmd + Shift + L** 選択したコードをコンテキストに追加
- **Cmd + K** インライン編集を開く
- **Cmd + Shift + K** 入力フォーカスの切り替え
- **Cmd + K** (ターミナルで) プロンプトバーを開く

公式: [Keyboard Shortcuts | Cursor Docs](https://cursor.com/ja/docs/configuration/kbd)
