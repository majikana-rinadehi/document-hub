
## 前提条件

- Claude Codeがインストール済み

### ステップ1：環境の準備


[https://github.com/oraios/serena?tab=readme-ov-file#quick-start](https://github.com/oraios/serena?tab=readme-ov-file#quick-start)

- uvのインストール
    - [https://docs.astral.sh/uv/getting-started/installation/#installation-methods](https://docs.astral.sh/uv/getting-started/installation/#installation-methods)
- uvxでserenaを実行

    ```plain text
    uvx --from git+https://github.com/oraios/serena serena start-mcp-server
    ```

    - →localhostでserenaダッシュボードが見れる [http://127.0.0.1:24282/dashboard/index.html](http://127.0.0.1:24282/dashboard/index.html)
- Claude Code にmcpとして追加

    ```plain text
    claude mcp add serena -- uvx --from git+https://github.com/oraios/serena serena start-mcp-server --context ide-assistant --project $(pwd)
    ```

    - 次回claudeコマンド実施すると、自動でlocalhostにserenaダッシュボードが表示
    - `/mcp` コマンドでserenaが追加されている
- プロジェクトのインデックス

    ```plain text
    uvx --from git+https://github.com/oraios/serena serena project index
    ```
