
Notion 記事編集のため、Claude Code のカスタムコマンドで Notion MCP を使って Notion 記事取得していたが、 Cursor に移行のため、Cursor でも 同 MCP を使えるようにする


## 結論

- `.cursor/mcp.json` に下記追加

```markdown
{
  "mcpServers": {
    "notionApi": {
      "command": "npx",
      "args": ["-y", "@notionhq/notion-mcp-server"],
      "env": {
        "OPENAPI_MCP_HEADERS": "{\"Authorization\": \"Bearer ntn_{token}\", \"Notion-Version\": \"2022-06-28\" }"
      }
    }
  }
}
```
