## 処理フロー図について

`processing-flow.md` には以下の内容が含まれています：

- **全体フロー**: Notionページ更新からGitHubへの保存、Zenn公開までの全体像
- **詳細フロー**: 各ステップの詳細な処理内容
- **コンポーネント構成**: システムを構成する各コンポーネントの関係
- **データフロー**: データがどのように流れるか
- **ステータス管理**: 処理の状態遷移

すべての図は Mermaid 形式で記述されており、GitHub や多くの Markdown ビューアーで自動的にレンダリングされます。
